# soolog

개발 블로그 프로젝트

## 노드버전 설정방법

이 프로젝트는 [`nvm`](https://github.com/nvm-sh/nvm)으로 노드 버전을 관리하고 있습니다.

```bash
nvm use
```

위 명령어로 프로젝트에 사용될 노드 버전을 사용 할 수 있습니다.

사용된 노드 버전: `v22.11.0`

## 실행 방법

```bash
npm i -g pnpm
pnpm dev
```
